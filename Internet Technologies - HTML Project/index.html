What is the Internet?

The Internet is a global network of interconnected computers that communicate using standardized protocols. It allows users to access and share information, communicate, and perform various tasks across vast distances almost instantly


How the Internet Works?

  • Client-Server Model
    • This is the backbone of Internet communication. A client (like your browser) sends a request to a server (where websites are hosted), and the server responds with the requested data



  • HTTP (HyperText Transfer Protocol)
    • The protocol used to transfer web pages. When you visit a website, your browser uses HTTP to request and receive content



  • DNS (Domain Name System)
    • Converts human-readable domain names (like example.com) into IP addresses that computers use to locate each other

What are Web Browsers?

Web browsers (like Chrome, Firefox, Edge) are software applications that retrieve, interpret, and display web content. They act as the client in the client-server model, sending HTTP requests and rendering HTML, CSS, and JavaScript


What is DNS and Why It's Important?

DNS is like the Internet's phonebook. It translates domain names into IP addresses. Without DNS, users would have to remember numeric IP addresses to visit websites. It ensures smooth navigation and accessibility across the web




Purpose of Each HTML Tag

HTML tags define the structure and content of a webpage. Each tag has a specific role

Tag Purpose
<html>
Root element of the page
<head>
Contains metadata, links to stylesheets, etc
<title>
Sets the page title in the browser tab
<body>
Contains the visible content
<h1> to <h6>
Headings, with <h1> being the most important
<p>
Paragraphs of text
<a>
Hyperlinks
<img>
Images
<ul>, <ol>, <li>
Lists (unordered and ordered)
<div>
Container for grouping content

When and How to Use tags

  • Use <h1> for the main title, and lower headings for subtopics
  • Use <p> for descriptive text
  • Use <a> to link to other pages or websites
  • Use <img> to display pictures
  • Use <div> to organize layout sections
  • Use <ul> or <ol> for bullet or numbered lists
